üìã Project Overview

Designed and implemented a scalable, secure enterprise network connecting five geographically distributed offices using advanced routing protocols and security technologies. The network serves a multinational organization with headquarters in Boston and Mumbai, and branch offices in New York, London, and Germany.

Project Scope:
  • 5 offices connected via centralized hub-and-spoke topology
  • Multi-area OSPF with Area 0 backbone + 5 stub areas
  • 24 end-user devices across 11 VLANs
  • Finance department isolation using extended ACLs
  • Automated IP management with DHCP and DNS services

üîÑ OSPF Multi-Area Design

The network implements hierarchical OSPF with a backbone (Area 0) and five stub areas for optimal scalability:

Area Type Location Router ID
Area 0 Backbone Core 0.0.0.0
Area 1 Standard Boston 1.1.1.1
Area 2 Standard Mumbai 2.2.2.2
Area 3 Standard London 3.3.3.3
Area 4 Standard New York 4.4.4.4
Area 5 Standard Germany 5.5.5.5
Design Rationale: The multi-area design localizes link-state database information within each area, reducing memory requirements and improving convergence time. All inter-area traffic transits through Area 0, ensuring loop-free routing.
