üîÆ GLITCHWITCH

XR Network Performance Analyzer for Quest 3 VR

Python Wireshark Analysis WiFi Monitoring Quest 3 VR AI-Powered PDF Reporting

Developed while researching network requirements for immersive communication systems at Northeastern's XR Lab. Traditional network monitoring tools don't understand how VR/XR applications experience network degradation‚ÄîGlitchWitch fills this gap by analyzing performance through the lens of immersive experience quality.

Research Question:

How do we evaluate if a network is ready for next-generation immersive communication technologies like VR collaboration, remote presence, and spatial computing?

Tool Capabilities:

  • Analyzes Wireshark packet captures for VR-specific metrics (latency <20ms, jitter <5ms, packet loss <0.5%)
  • Calculates "VR Readiness Score" (0-100) based on thresholds that affect immersive communication
  • Identifies exact timestamps where network glitches would break presence in VR collaboration
  • Correlates WiFi signal strength with packet performance degradation
  • Generates professional PDF reports with AI-powered interpretation
  • Predicts network conditions that cause motion sickness or communication breakdowns

Research Impact:

This research addresses a critical gap in telecommunications: as communication moves from 2D screens to immersive 3D spaces, our infrastructure evaluation methods must evolve. GlitchWitch demonstrates that VR collaboration systems require fundamentally different network performance criteria than traditional video conferencing or data applications.
